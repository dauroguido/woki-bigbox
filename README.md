# Bigbox × Woki — Pieza de comunicación cruzada

Email + landing page que Bigbox manda a su base de restaurantes prestadores en Argentina para comunicar la integración con Woki.

**Frase ancla:** Bigbox trae nuevos clientes. Woki te ayuda a mejorar su experiencia y fidelizarlos.

## Entregables

| Archivo | Descripción |
|---|---|
| [`landing.html`](landing.html) | Landing page single-file. Track B2B (azul Woki), 10 secciones, Mona Sans + Figtree, mockup HTML/CSS del panel de reservas. |
| [`email.html`](email.html) | HTML email-friendly (tablas, estilos inline, ancho 600px, websafe fallbacks, mobile responsive). |
| [`email.txt`](email.txt) | Versión texto plano del email para clientes que no soportan HTML. |
| [`handoff_bigbox_x_woki.md`](handoff_bigbox_x_woki.md) | Documento de brief / fuente de verdad. |
| `assets/logos/` | Logos Woki B2B (solid, gradient, white) + logo Bigbox provisional. |
| `assets/fonts/` | Figtree + Mona Sans variables. |
